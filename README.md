# CloudFlare 部署自己的密码管理器

一、部署流程：

1.1、fork 本项目

1.2、到cloudflare 新建pages，点击git导入，然后选择你fork这个项目的项目名字

1.3、开始部署，第一次部署完成后，去cloudflare-pages-项目中的“设置”找到变量绑定

=========================================变量绑定

1.4、变量填：
            username1= 你的用户

            passsword1=你的密码
          
1.5、KV库，变量名=PASSWORD_KV
